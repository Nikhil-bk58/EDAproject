## What is EDA?

Exploratory Data Analysis (EDA) is the process of analyzing datasets to summarize their main characteristics.
It uses visual methods like charts and graphs to understand the data.
EDA helps in detecting patterns, trends, and outliers.
It is an important step before applying machine learning models.
EDA improves decision-making by providing insights from raw data.
